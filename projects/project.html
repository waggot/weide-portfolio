about this project


Shark-san is a bot inspired by Ikea's soft toy shark. Shark-san is made using Discord.Js. Shark-san has 11 features that I have programmed with Discord.js and also made a simple website to showcase some of them. Shark-san is a utility bot that is made with the goal to help lighten the server's mood with features like coinflip and send blahaj images to the server. My goal for the project is to have many more fun commands and most importantly an economy system. Being new to programming, I lack the knowledge to make such commands and hope that i will be able to have a deeper understanding in programming and have the ability to make such features.
