about this project


This project is made for my eae application as my portolfio to showcase the stuff i have programmed. I tried to keep UI to be simple and clean using black and white and also added animations to make the website more lively. Since I started this project I have made 3 drafts and have come a long way with my final design.
I used HTML and CSS for the styling of the website and experimented on many scroll animation libraries before settling with Sal.js. Through the Github Student developer kit, I have gotten my own domain and hosted the website on github.

Technologies used


  • HTML5
  • CSS3
  • Javascript
  • Node.js
  • Sal.js
  • A.O.S
